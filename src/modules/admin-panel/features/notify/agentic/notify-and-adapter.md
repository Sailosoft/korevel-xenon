# AdminPanelNotify

- It can be a standalone snackbar
- It can be integrated with any hook that implements AdminPanelNotify interface
- Receives as well differ type
  - accent
  - default
  - success
  - warning
  - error
  - custom

# AdminPanelNotifyAdapter

- AdminPanelNotifyAdapter is a function that can be used to integrate AdminPanelNotify with any hook that implements AdminPanelNotify interface
- Notify can be varied if you use your own self made snackbar or use hero ui toast.
