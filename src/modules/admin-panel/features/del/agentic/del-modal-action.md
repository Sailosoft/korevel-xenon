# Delete modal and Action

## Dependencies

- AdminPanelMutation
- AdminPanelTable
- AdminPanelNotify

## Features

- Manage delete modal state
- Manage delete action using the D operation from AdminPanelMutation.Delete
- Reload the data from the table after successful but keep the query options.
- Close the modal after successful delete action.
- Show a snackbar after successful delete action.
- Dependencies are inject on the parameters
