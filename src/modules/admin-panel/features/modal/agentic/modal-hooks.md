# UseAdminPanelModal hooks

- Maintain a modal state that can be refer by AdminPanelForm and Trigger By Admin Panel Table

## Features

- Supports loading when data is being fetched
- Supports modal event type such create, update, view, and plain.
- Modal State such as open, close, loading, mode.

## Events

- Create event in table: Trigger a modal
- Update event in table: Trigger a modal
- View event in table: Trigger a modal

## Usage

- For open modal
- For managing mode

## Rules

- It does not handle data and loading of data
- It does not handle mutation operation, it only open and close the modal
